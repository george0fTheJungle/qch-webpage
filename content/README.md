# Content Drop Zone

Place your site content here for Claude to use when building the site.

## What to add:

### 1. `copy.md` — All your website text
Create a file called `copy.md` with your site text organized by section:

```markdown
# Hero Section
Your main headline here
Your subtitle/tagline here

# About / Who We Are
Paragraph text about your company...

# Services
## Service 1 Name
Description...

## Service 2 Name
Description...

# Contact
Phone: ...
Email: ...
Address: ...

# Footer
Any footer text...
```

### 2. Images → `public/images/`
Drop all your images (photos, logo, etc.) into the `public/images/` folder.

Name them descriptively so I know where to use them:
- `logo.png` — Your company logo
- `hero.jpg` — Hero/banner image
- `about.jpg` — About section image
- `service-1.jpg`, `service-2.jpg` — Service images
- `project-1.jpg`, `project-2.jpg` — Project/portfolio images
- etc.

### 3. `brand.md` (optional) — Brand preferences
If you have specific brand colors, fonts, or style preferences:

```markdown
# Brand
Primary color: #hex
Secondary color: #hex
Font preference: modern/classic/etc.
Vibe: luxury/industrial/clean/bold/etc.
```
